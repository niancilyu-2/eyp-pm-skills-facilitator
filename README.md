# eyp-pm-skills — facilitator materials (private)

Companion to the public repo [`niancilyu-2/eyp-pm-skills`](https://github.com/niancilyu-2/eyp-pm-skills).
This repo holds everything that would spoil the exercises if participants (or their agents,
which read the working directory) could see it. Keep it private.

| File | What it is |
|------|-----------|
| `FACILITATOR_GUIDE.md` | Run-of-show: prompts, timings, expected outputs, talking points, fabrication test |
| `ANSWER_KEY.md` | The seeded scenarios and expected numbers for both datasets |
| `sample-outputs/` | Captured artifacts from a full run of the chain — live fallbacks for every stage |

Before a session: clone this next to (not inside) the public repo clone. The public repo stays
clean; open the guide and fallbacks from here.
